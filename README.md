### ðŸ”§ For Developers (Advanced)

If you want to work with the source code:

1. Clone the repository:
    ```bash
    git clone https://github.com/Groot-Made/Connector.git wp-content/plugins/grootmade-connector
    ```

2. Set up your environment:
    - Copy `.env.example` to `.env`
    - Update the values in `.env` as needed

3. Install dependencies:
    ```bash
    npm install
    composer install
    ```

4. Build the plugin:
    ```bash
    npm run build
    ```
    Or for development mode with live reloading:
    ```bash
    npm start
    ```
